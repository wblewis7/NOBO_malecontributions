# NOBO_malecontributions
Formatted data and R/JAGS code for assessing proportional male contributions to demographics of northern bobwhite (Colinus virginianus) in the Southeastern U.S. From: Roberts, J. G., J. A. Rectenwald, D. C. Sisson, A. S. Cramer, H. K. Lott, M. A. Hazelbaker, W. B. Lewis, and J. A. Martin. Exploring male contribtuions to Northern bobwhite demographics.
